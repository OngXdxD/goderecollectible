How to Change Logo ?

Go To "public/assets/images/brand-logos" folder and replace your logo with Previous Logos within in image size. note: Please don't increase logo sizes. Replace your logo within given image size. otherwise the logo will not fit in particular place it disturbs the template design.

How To Disable Switcher In All Pages ?

Step1:

Open header.tsx component shared\layout-components\header\header.tsx

To remove switcher section as shown below.

Step2:

Remove the switcher component from the main layout follow the path shared\layout-components\layout\content-layout.tsx

Step3:

Now remove the switcher component and switcherData file from the root folder, follow the path Switcher component shared\layout-components\switcher\switcher.tsx and switcherdata file from shared\switcherdata\switcherdata.tsx

Note : After completing above steps please run the command to update dist folder.
