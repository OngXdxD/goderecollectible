2) Updated Plugins And Added New Plugins. 
----------------------------------------------------------------------
	 Several packages have been updated to their latest versions to ensure compatibility and security.
	
	To get the latest packages, please copy the content of the provided package.json file, paste it into your existing package.json, and run npm install.

3) Added New Switcher features
----------------------------------------------------------------------
	New switcher is added to Nowa
	In this switcher we have added new features like 

	1) Single switch Rtl 
	2) Single switch menus
	3) Changing colors 
	4) Dark Theme and many more 

	For a more comprehensive understanding of Switcher ,we strongly recommend reviewing our updated documentation. 

	To add swicher page replace complete assets folder replace switcher content in all NextJs pages.
